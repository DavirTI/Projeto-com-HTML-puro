Oque são clientes?

Clientes no contexto da informática são dispositivos que realizam requisições a servidores, como um usuário que ao clicar na barra de endereços no navegador busca pelo site do Google ou mesmo ao enviar uma mensagem pelo WhatsApp, podemos utilizar a analogia de um cliente que faz um pedido em uma loja de café, ele faz a solicitação do seu pedido, e prontamente o servidor (loja de café) atende a requisição, requisições são feitas o tempo todo na internet pelos clientes afim de obterem algum serviço.


Oque são servidores?

Servidores são dispositivos que atendem requisições de clientes, como um garçom que atende pedidos de clientes em uma loja de café, o servidor é responsável por atender as requisições dos clientes, como por exemplo, ao acessar o site do YouTube o servidor do YouTube atende a requisição do cliente e devolve a página do YouTube para o cliente, servidores são dispositivos que estão sempre ligados e prontos para atender as requisições dos clientes.


Linguagem de programação VS Linguagem de marcação

Linguagem de programação é uma linguagem que permite a criação de algoritmos e programas, como por exemplo, a linguagem de programação Python que é uma linguagem de programação muito utilizada para desenvolvimento de aplicações web, já a linguagem de marcação é uma linguagem que permite a marcação de texto, como por exemplo, a linguagem de marcação HTML que é uma linguagem de marcação muito utilizada para criação de páginas web.


Estrutura básica de um HTML

A estrutura básica de um HTML é composta por TAGS, que são elementos que compõem a estrutura de uma página web, como por exemplo, a TAG <html> que é a tag raiz de um documento HTML, a TAG <head> que é a tag que contém informações sobre o documento, como por exemplo, o título da página, a TAG <body> que é a tag que contém o conteúdo do documento, como por exemplo, textos, imagens, vídeos, etc.


Atributos das TAGS

Atributos são propriedades que podem ser adicionadas a uma TAG, como por exemplo, a TAG <img> que é a tag que exibe uma imagem em uma página web, ela possui o atributo src que é o atributo que contém o caminho da imagem, a TAG <a> que é a tag que cria um link em uma página web, ela possui o atributo href que é o atributo que contém o endereço do link.
